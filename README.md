## Description

Developed an interactive Microsoft Excel dashboard to analyze 2015 U.S. sales performance using Pivot Tables, Pivot Charts, Slicers, and Excel formulas.

## Key Features

- Sales by Product Category
- Sales by Product Sub-Category
- Sales by Customer Segment
- Top 10 Customers by Profit
- Regional Sales Analysis
- Interactive Slicers and Filters

## Tools

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Analysis
